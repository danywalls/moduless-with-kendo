# Improve Angular Performance with Lazy Loading, @defer and Kendo UI

## Project Summary
A guide on optimizing Angular application performance using Lazy Loading, Deferrable Views (`@defer`), and Kendo UI components for loading indicators.

## How to Run
### Installation
1. `npm install` 

### Running
1. `ng serve` 
2. Use the network throttling in DevTools to see the `@defer` triggers in action.
